# 🤖 AI Customer Retention Assistant

An end-to-end Machine Learning + AI Agent system that predicts customer churn and recommends business actions.

---

## 🚀 Project Overview

This project simulates a real-world business use case where companies proactively identify customers at risk of churn and take action to retain them.

The system combines:
- Machine Learning model for prediction
- FastAPI backend for serving predictions
- AI agent for decision-making
- Gradio UI for interactive demo

---

## 🧠 Architecture

User → UI (Gradio) → Chatbot Agent → FastAPI → ML Model → Response → UI

---

## ⚙️ Features

- 🔍 Customer churn prediction using ML model
- 📊 Confidence score for predictions
- 🤖 AI agent suggesting retention strategies
- 🌐 REST API using FastAPI
- 🎨 Interactive UI using Gradio
- 🧩 Modular architecture (API + Agent + UI)

---

## 📦 Tech Stack

- Python
- Scikit-learn
- FastAPI
- Gradio
- NumPy / Pandas
